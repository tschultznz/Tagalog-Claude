# Tagalog Tutor Pre-clean Package

This package is a reversible, normalized handoff for research and system design.

## Use first

Read in this order:

1. `planning/AUDIT.md`
2. `current/progress.txt`
3. `current/system_prompt_tom_tagalog_tutor.txt`
4. `current/curriculum_stage4b_modules_51_70.txt`
5. `legacy/curriculum_modules_1_50.txt`
6. `corpus/tagalog_sentences.txt`

## Rules

- Do not edit the raw corpus.
- Do not treat legacy prompts as active instructions.
- Preserve source provenance.
- Do not begin major implementation until the learner model, skill graph, and scheduling schema are agreed.
- Treat Modules 1–50 as historical learning coverage, not proof of current mastery.
